# Trabajo Práctico Especial - Programación 3

## Descripción

Este proyecto resuelve un problema de planificación de producción en una fábrica de autopartes. Dado un número de piezas a producir y una lista de máquinas con su producción fija, se busca minimizar la cantidad de puestas en marcha utilizando:

- Backtracking (óptimo con poda)
- Greedy (rápido pero no siempre óptimo)


El archivo se llama config.txt y contiene lo siguiente:
12
M1,7
M2,3
M3,4
M4,1
